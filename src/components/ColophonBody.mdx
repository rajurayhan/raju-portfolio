# Colophon

This site recreates the **Raju Rayhan** handoff from Claude Design: warm paper / ink, JetBrains Mono + Instrument Serif, and the full casebook including six long-form case studies.

- **Astro** for static pages and routing  
- **Vue** islands for the filterable work index and the contact form  
- **This MDX fragment** (you are reading it) for portable rich text in components  
- **Cloudflare** adapter for **Pages**; contact posts go to `POST /api/contact` and email via **Resend**

Fonts load from the main stylesheet (Google Fonts, same as the prototype).
